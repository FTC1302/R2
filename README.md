== R2 ==

---- program for robot R2 of FTC team 1302 ----

FTC参赛队 1302 2013赛季：Ring it UP!!! 参赛机器人R2 程序。

/drivers 		: 角度传感器、红外线传感器及各种其他hitechnic传感器驱动程序所在目录
/prototype_drivers 	: 原型板驱动
R2_const.h		: 常数定义文件
R2_lib.h		: 库函数定义文件
R2_TELEOP.C 		: R2遥控阶段正式程序
